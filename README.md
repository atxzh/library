# Library

A Digital Library Markup

## Features

- [x] Add New Books
- [x] Delete Books
- [x] Edit Existing Books
    - [x] Create Book Logs
- [x] Filter & Sorting
    - Filter by:
        - Search
        - Tags
    - Sort by Types in DropDown
- [] Local Storage 
- [] Login with Firebase
